Mac常用快捷键

2023-09-08 小姬炖蘑菇

Mac常用快捷键

command + W 关闭当前标签页

Command + Q 退出当前程序

option+commad+esc 强制退出

Command + M 最小化窗口

Command + H 隐藏窗口

Command+option+H 除当前窗口隐藏其他窗口

Command + , 打开偏好设置

Command + tab 切换程序

Command + tab + ~ 反向切换程序

Command + tab + Q 切换程序同时退出

Control + 上下左右键 模拟触控板三指操作（好用）

访达文件管理

Command/shift + 光标 多选文件

Command + I 查看文件信息

Command + J 设置选项

选中文件 + Enter 重命名

Command + T 访达中新建标签页

Command + N 访达中新建窗口

Command + 双击文件夹 新建窗口打开

Command + 上下键 到达上级文件夹，进入文件夹

Command + delete 文件移到废纸篓

Fn + C 打开常用开关

浏览器

Control + tab 切换标签页

Control + shift + tab 反向切换标签

Command + 12345 精准切换标签页

Command + W 关闭标签页

Command + shift + t 恢复关闭的网页

Command + Q 完全关闭浏览器

Option + 上下键 整页翻页

Command + 上下 直达页低/页顶

Command + ‘+-‘ 放大缩小网页

Command + 0 恢复正常网页

Command + R 刷新网页

Command + [ 返回上个网页

Command + ] 前往下个网页
